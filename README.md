## üìã Available Commands

| Command | Description |
|---------|-------------|
| `help` | Display all available commands |
| `about` | Learn about me and my background |
| `skills` | View my technical skills and expertise |
| `projects` | Browse my featured projects |
| `contact` | Get my contact information |
| `links` | List all available social links |
| `open [link-name]` | Open a specific link in new tab |
| `surprise` | Display random ASCII art |
| `clear` | Clear the terminal screen |

## üõ†Ô∏è Customization Guide

### 1. Update Personal Information

**File: `commands/about.html`**
- Change your name
- Update your bio and description
- Modify your tech stack
- Add your personal touch

### 2. Update Contact Details

**File: `commands/contact.html`**
- Replace email address
- Update social media links
- Add/remove contact methods

### 3. Showcase Your Projects

**File: `commands/project.html`**
- Add your project titles
- Write compelling descriptions
- Highlight key technologies used
- Link to GitHub repos and live demos

### 4. Update Skills

**File: `script.js` (line ~30)**
```javascript
const technologies = [
    ["JavaScript", "TypeScript", "Python", "Go"],
    ["PostgreSQL", "MongoDB", "Redis", "MySQL"],
    ["Docker", "Kubernetes", "AWS", "CI/CD"],
    ["React", "Vue", "Node.js", "Express"],
    ["Git", "Linux", "Nginx", "GraphQL"]
];
```

### 5. Update Links

**File: `script.js` (line ~50)**
```javascript
const links = {
  "github": "your-github-url",
  "linkedin": "your-linkedin-url",
  "twitter": "your-twitter-url",
  "portfolio": "your-portfolio-url"
};
```

### 6. Customize Colors

**File: `styles.css` (line ~3)**
```css
:root {
    --bg-primary: #0a0e27;
    --accent-primary: #00ff9f;
    --accent-secondary: #00d4ff;
    /* Modify these to match your brand */
}
```

### 7. Update ASCII Art

**File: `script.js` (line ~8)**
- Replace the ASCII banner with your name
- Use [patorjk.com](https://patorjk.com/software/taag/) to generate custom ASCII art
